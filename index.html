🔄 Luồng dữ liệu chính (Key Data Flows)
1
GT POS → Kafka → Data Warehouse
Mỗi giao dịch tại tạp hoá stream real-time vào data lake. Tất cả brand, không chỉ MCH.
2
Data Warehouse → Recommendation Engine
ML model train trên purchase history + regional patterns → gợi ý bữa cơm cá nhân hoá.
3
Reco Engine → Zalo OA API → Khách hàng
Gợi ý gửi qua Zalo OA tin nhắn. Cần consent opt-in khi đăng ký WiN Member.
4
GT POS signup → Member Service → Zalo OA follow
Khách đăng ký WiN tại tạp hoá → auto-follow Zalo OA → nhận gợi ý. Đây là acquisition loop.
5
WinX Ads Engine → Promotion Engine → POS
Brand tạo campaign → inject vào recommendation → đo attribution bằng POS sell-through.
6
GT POS data → Credit Scoring → Bank Lending API
Purchase frequency + revenue stability → credit score → ngân hàng đối tác auto-approve thấu chi.
